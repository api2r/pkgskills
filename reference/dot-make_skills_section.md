# Build a complete \## Skills section with a table

Build a complete \## Skills section with a table

## Usage

``` r
.make_skills_section(row)
```

## Arguments

- row:

  (`character(1)`) A pre-built skill row, e.g. from
  [`.make_skill_row()`](https://api2r.github.io/pkgskills/reference/dot-make_skill_row.md).

## Value

(`character`) Lines comprising the section heading and table.
