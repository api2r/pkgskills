# Build the header and data rows for a Skills markdown table

Build the header and data rows for a Skills markdown table

## Usage

``` r
.make_skills_table(row)
```

## Arguments

- row:

  (`character(1)`) A pre-built skill row, e.g. from
  [`.make_skill_row()`](https://pkgskills.api2r.org/reference/dot-make_skill_row.md).

## Value

(`character`) Lines comprising the blank spacer, table header,
separator, and data row.
