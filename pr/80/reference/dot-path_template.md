# Build a path to a file in `inst/templates/`

Build a path to a file in `inst/templates/`

## Usage

``` r
.path_template(...)
```

## Arguments

- ...:

  Path components appended after `"templates"`, passed to
  [`.path_pkg()`](https://pkgskills.api2r.org/reference/dot-path_pkg.md).

## Value

(`character(1)`) Absolute path to the template file.
