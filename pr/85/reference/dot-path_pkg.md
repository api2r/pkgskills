# Build a path to a inst file within installed or dev pkgskills

Build a path to a inst file within installed or dev pkgskills

## Usage

``` r
.path_pkg(...)
```

## Arguments

- ...:

  Path components, passed to
  [`fs::path_package()`](https://fs.r-lib.org/reference/path_package.html).

## Value

(`character(1)`) Absolute path within the pkgskills package.
