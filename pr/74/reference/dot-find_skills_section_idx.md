# Find the line index of the \## Skills section heading in AGENTS.md

Find the line index of the \## Skills section heading in AGENTS.md

## Usage

``` r
.find_skills_section_idx(agents_lines)
```

## Arguments

- agents_lines:

  (`character`) Lines of `AGENTS.md`.

## Value

(`integer(1)`) Index of the `## Skills` heading, or
[`integer()`](https://rdrr.io/r/base/integer.html).
