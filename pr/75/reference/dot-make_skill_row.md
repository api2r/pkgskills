# Build a markdown table row for a skill

Build a markdown table row for a skill

## Usage

``` r
.make_skill_row(trigger, save_as, call = caller_env())
```

## Arguments

- trigger:

  (`character(1)`) Trigger phrase for the skill.

- save_as:

  (`character(1)`) Output file path, relative to the project root.

- call:

  (`environment`) The caller environment for error messages.

## Value

(`character(1)`) A markdown table row string.
